## Git Stages

----------

### Git Workflow

- Work on something:
  - ignoring
  - Add to Index
  - Commit and write message
  - Push to Remote

----------

### Git Workflow

- Get stuff from the server
  - Fetch  updates
  - Pull to master
  - Merge into ...
  - Rebase master to your branch

----------

### Eclipse Views
- Repository view
  - Manage repos
- Staging view
  - Workflow: Index, Commits and Push
- History View
  - Audit
  - Review branches
